# 《程序员生存手册》
《程序员生存手册》分别从技能篇、方法篇、发展规划篇、资源篇、管理篇五个部分来介绍相关的知识，涵盖了每位程序员从初入行业到准备转型，直至最终实现自己人生价值的全过程。希望能为阅读到这本手册的每一位程序员提供一些提高能力的新途径，以及进行职业规划的新思路。

github地址为：`https://github.com/easysoft/zendevelopper`
